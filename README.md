# Setup Dev

To launch mkdocs locally, follow these instructions:

1. Create a Python environment:
```bash
python3 -m venv .venv
. .venv/bin/activate
```

1. Install the dependencies:
```bash
make install
```

1. Start the serving endpoint:
```bash
make serve
```
